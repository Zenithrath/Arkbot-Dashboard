## Step-by-Step Setup di n8n

### Step 1: Buat Workflow Baru
1. Login ke n8n (`https://arkbot-n8n.6jkqbm.easypanel.host`)
2. Klik **New Workflow**
3. Beri nama: `Google Drive Manager`

### Step 2: Tambah Node Webhook
1. Klik **+** → cari **Webhook**
2. Settings:
   - **Method**: `POST`
   - **Path**: `drive-manager`
   - **Response Mode**: `Last Node` ⚠️ PENTING!
3. Catat URL: `https://arkbot-n8n.6jkqbm.easypanel.host/webhook/drive-manager`

### Step 3: Tambah Node Switch
1. Klik **+** setelah Webhook → cari **Switch**
2. Settings:
   - **Mode**: Rules
   - **Rules**:
     - **Rule 1**: 
       - Type: `String`
       - Value 1: `{{ $json.action }}`
       - Operation: `equals`
       - Value 2: `list`
     - **Rule 2**:
       - Type: `String`
       - Value 1: `{{ $json.action }}`
       - Operation: `equals`
       - Value 2: `delete`

### Step 4: Tambah Node Google Drive (List)
1. Klik **+** dari output 0 Switch → cari **Google Drive**
2. Settings:
   - **Credential**: Pilih Google OAuth2 credential yang sudah ada
   - **Operation**: `Get Many`
   - **Return All**: `true`
3. Rename node: `Google Drive - List`

### Step 5: Tambah Node Set (Format List Response)
1. Klik **+** setelah Google Drive List → cari **Set**
2. Settings:
   - **Mode**: Manual Mapping
   - **Fields**:
     - Name: `files`
     - Type: `Array`
     - Value: `={{ $json.files }}`
3. Rename node: `Format List Response`

### Step 6: Tambah Node Google Drive (Delete)
1. Klik **+** dari output 1 Switch → cari **Google Drive**
2. Settings:
   - **Credential**: Pilih Google OAuth2 credential yang sama
   - **Operation**: `Delete`
   - **File ID**: `={{ $('Webhook').item.json.drive_file_id }}`
3. Rename node: `Google Drive - Delete`

### Step 7: Tambah Node Set (Format Delete Response)
1. Klik **+** setelah Google Drive Delete → cari **Set**
2. Settings:
   - **Mode**: Manual Mapping
   - **Fields**:
     - Name: `success`
     - Type: `Boolean`
     - Value: `true`
3. Rename node: `Format Delete Response`

### Step 8: HAPUS Node Respond to Webhook
1. Cari semua node **Respond to Webhook**
2. Klik node → klik tombol **Delete** (atau tekan Delete key)
3. ⚠️ JANGAN biarkan ada node Respond to Webhook yang tidak tersambung!

### Step 9: Connect Nodes
```
Webhook → Switch
Switch output 0 → Google Drive - List → Format List Response
Switch output 1 → Google Drive - Delete → Format Delete Response
```

### Step 10: Save & Activate
1. Klik **Save**
2. Toggle **Active** ke on

## Troubleshooting

### Error "Unused Respond to Webhook"
- Hapus SEMUA node "Respond to Webhook" yang ada
- Pastikan Webhook Response Mode = "Last Node"

### Error 403 Forbidden
- Cek Google OAuth2 credential sudah benar
- Re-authenticate jika perlu

### Error "Cannot find module"
- Pastikan Google Drive node sudah install
- Cek versi n8n minimal v1.0
