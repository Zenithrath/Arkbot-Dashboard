# n8n Workflow: Google Drive Manager (Gabungan)

## Workflow Structure

```
Webhook → Switch → [List Files] / [Delete File]
```

## Step-by-Step Setup

### 1. Buat Workflow Baru di n8n

### 2. Tambah Node: Webhook
- **Method**: POST
- **Path**: `/drive-manager`
- **Response Mode**: `Last Node` (bukan "Using Respond to Webhook node")

### 3. Tambah Node: Switch
- **Type**: Switch
- **Rules**:
  - Rule 1: `{{ $json.action }}` equals `list` → output 0
  - Rule 2: `{{ $json.action }}` equals `delete` → output 1

### 4. Tambah Node: Google Drive (List)
- **Type**: Google Drive
- **Operation**: Get Many
- **Authenticate**: Google OAuth2
- **Return All**: true
- **Connect**: dari Switch output 0

### 5. Tambah Node: Set (Format List Response)
- **Type**: Set
- **Values**:
  - `files`: `={{ $json.files }}`
- **Connect**: dari Google Drive (List)

### 6. Tambah Node: Google Drive (Delete)
- **Type**: Google Drive
- **Operation**: Delete
- **File ID**: `={{ $('Webhook').item.json.drive_file_id }}`
- **Connect**: dari Switch output 1

### 7. Tambah Node: Set (Format Delete Response)
- **Type**: Set
- **Values**:
  - `success`: `true`
  - `message`: `File deleted`
- **Connect**: dari Google Drive (Delete)

### 8. Hapus semua node "Respond to Webhook" yang ada

### 9. Activate Workflow

## Cara Pakai

### List Files
```json
POST /webhook/drive-manager
{
  "action": "list"
}
```

Response:
```json
{
  "files": [
    { "id": "xxx", "name": "file.pdf", "mimeType": "...", "size": "123", "modifiedTime": "..." }
  ]
}
```

### Delete File
```json
POST /webhook/drive-manager
{
  "action": "delete",
  "drive_file_id": "file_id_dari_google_drive"
}
```

Response:
```json
{
  "success": true,
  "message": "File deleted"
}
```

## Frontend Update

**File:** `src/pages/DocumentsPage.tsx`

Ganti dua URL jadi satu:
```typescript
const N8N_DRIVE_MANAGER_URL =
  "https://arkbot-n8n.6jkqbm.easypanel.host/webhook/drive-manager"
```

Update fetch functions:
```typescript
// List files
const res = await fetch(N8N_DRIVE_MANAGER_URL, {
  method: "POST",
  body: JSON.stringify({ action: "list" })
})

// Delete file
const res = await fetch(N8N_DRIVE_MANAGER_URL, {
  method: "POST",
  body: JSON.stringify({ action: "delete", drive_file_id: fileId })
})
```
