# Arkbot Dashboard - Google Drive Files Viewer Plan

## Ringkasan
Menambahkan fitur untuk melihat file di Google Drive via n8n webhook, dengan kemampuan hapus file dari Drive untuk handle file yang duplikat.

---

## 1. Tambah n8n Webhook Endpoints

### A. List Files from Google Drive
**n8n Workflow Steps:**
1. **Webhook** - POST `/webhook/list-drive-files`
2. **Google Drive API** - List files di folder yang ditentukan
   - Node: "Google Drive" → Operation: "Get Many"
   - Filter: MIME type = `application/pdf`
   - Return: `{ files: [{ id, name, mimeType, size, modifiedTime }] }`

### B. Delete File from Google Drive
**n8n Workflow Steps:**
1. **Webhook** - POST `/webhook/delete-drive-file`
2. **Google Drive API** - Delete file
   - Node: "Google Drive" → Operation: "Delete"
   - Input: `{{ $json.drive_file_id }}`
   - Return: `{ success: true }` atau error

## 2. Frontend Implementation

### A. Tambah Tab/Section di Documents Page
**File:** `src/pages/DocumentsPage.tsx`

**Tab 1: Database Files** (yang sudah ada - file dari `drive_file_sync`)
**Tab 2: Google Drive Files** (baru - file langsung dari Drive via n8n)

**Interface:**
```typescript
interface DriveCloudFile {
  id: string
  name: string
  mimeType: string
  size: string
  modifiedTime: string
}
```

**State:**
```typescript
const [driveFiles, setDriveFiles] = useState<DriveCloudFile[]>([])
const [driveLoading, setDriveLoading] = useState(false)
const [activeTab, setActiveTab] = useState<"database" | "drive">("database")
```

**Fetch Function:**
```typescript
const fetchDriveFiles = async () => {
  setDriveLoading(true)
  try {
    const res = await fetch(N8N_LIST_DRIVE_URL, {
      method: "POST",
      headers: { "Content-Type": "application/json" },
    })
    const data = await res.json()
    setDriveFiles(data.files || [])
  } catch (err) {
    toast.error("Gagal mengambil file dari Google Drive")
  }
  setDriveLoading(false)
}
```

**Delete Function:**
```typescript
const handleDeleteDriveFile = async (fileId: string, fileName: string) => {
  try {
    const res = await fetch(N8N_DELETE_DRIVE_URL, {
      method: "POST",
      headers: { "Content-Type": "application/json" },
      body: JSON.stringify({ drive_file_id: fileId }),
    })
    if (res.ok) {
      toast.success(`${fileName} berhasil dihapus dari Drive`)
      setDriveFiles(prev => prev.filter(f => f.id !== fileId))
    }
  } catch (err) {
    toast.error("Gagal menghapus file dari Drive")
  }
}
```

## 5. n8n Workflow Setup (User Guide)

### Workflow 1: List Drive Files
1. Buka n8n → Buat workflow baru
2. Tambah node **Webhook**:
   - Method: POST
   - Path: `/list-drive-files`
3. Tambah node **Google Drive**:
   - Operation: Get Many
   - Authenticate: Google OAuth2
   - Return All: true
   - Filter: MIME type = `application/pdf`
4. Tambah node **Respond to Webhook**:
   - Response: `{{ { files: $json } }}`
5. Activate workflow

### Workflow 2: Delete Drive File
1. Buka n8n → Buat workflow baru
2. Tambah node **Webhook**:
   - Method: POST
   - Path: `/delete-drive-file`
3. Tambah node **Google Drive**:
   - Operation: Delete
   - File ID: `{{ $json.drive_file_id }}`
4. Tambah node **Respond to Webhook**:
   - Response: `{{ { success: true } }}`
5. Activate workflow
